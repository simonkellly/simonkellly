### Speedcubing

- **[Speedcubing Ireland Website](https://www.speedcubingireland.com/)** - Official website for Speedcubing Ireland, the World Cube Association's Irish regional organisation.

- **[BLD Ninja](https://simonkellly.github.io/bld-ninja/)** - Trainer for 3x3 blindfolded using bluetooth speedcubes to enable solve analysis and case training.

- **[BTCube Web](https://www.npmjs.com/package/btcube-web)** - Typescript library allowing connecting and integrating multiple brands of Bluetooth speedcubes with browser Web Bluetooth.

- **[WCA Scramble Password Organiser](https://simonkellly.github.io/scramble-organizer/)** - Automatic sorting tool for scramble passwords for World Cube Association comeptitions.

- **[Competitor Identifier](https://simonkellly.github.io/competitor-identifier/)** - Utility for identifying and looking up World Cube Association competitor IDs for specific competitions.

- **[WCA Statistics Ireland](https://simonkellly.github.io/wca_statistics_ireland/)** - A fork of WCA Statistics showing records and statistics only considering Irish competitors.

- **[Simultaneous Scrambling](https://simonkellly.github.io/simultaneous-scrambling/)** - WCA Scramble viewer for running multiple events at the same time

- **[TypeComp](https://github.com/simonkellly/TypeComp)** - Typescript library for WCA competition activity assignments

### Anki & Studying

- **[Deck Browser CSS](https://ankiweb.net/shared/info/153779542)** - Anki Add-on allowing for easy customisation of the application's style sheets.

- **[Side Cards](https://github.com/simonkellly/side-cards)** - Work-in-progress Obsidian plugin for creating and managing flashcards which are co-located with the note content, and can be exported to tools such as Anki.
